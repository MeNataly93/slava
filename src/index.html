  • Barnefest fotograf

    Vår fotograf kan fange viktige øyeblikk og følelsar hjå barn og deira foreldre, og lage kvalitetsrike og minneverdige fotografi.

    500 kr pr 1 time

  • Såpeboble

    Ein såpeboblegenerator er ein innretning som blir brukt til å lage såpebobler. Dei blir ofte brukt til leik og underhaldning.

    400 kr

  • Glittertatoveringen

    Glittertatoveringar er midlertidige tatoveringar som brukast med lim og glitter. Dei finst i ulike design og er populære til festleg huddekorasjon.

    Fest: 1000 kr pr 1 time

    Bursdag: 500 kr pr 1 time (20 barna)

  • Ansiktsmaling

    Akvamaling skaper midlertidige teikningar på huda, er trygt, lett å påføre, og vaskast bort med vatn.

    Fest: 1000 kr pr 1 time / 50-100 kr pr ein person

    Bursdag: 500 kr pr 1 time (ca. 8 barn)
